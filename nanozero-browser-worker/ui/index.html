Calibre ta machine

Mesure le débit réel de ta machine pour régler Éco / Normal / Max.

Comment ça marche ?

Mode Max : ta machine va chauffer et ventiler davantage.

Sur batterie ? Le mode Éco ménage ton autonomie.

Peu de cœurs CPU détectés : la contribution sera lente (le GPU reste utilisé s'il est disponible).

Avancé ⚠️ Plus de workers = plus de RAM/sessions dans l'onglet (mémoire GPU sur GPU). Le calibrage borne K au maximum tenable de ta machine ; reviens en arrière si l'onglet rame.
